== Description ==

Convert WordPress publication dates in your posts and pages to the current stardate.  Also adds a custom function and a few shortcodes for use in themes and the post editor.

Get your geek on!

For more information on the WordPress <a href="http://seoserpent.com/wordpress/stardate">Stardate</a> plugin and how to use it, visit the information page at the preceding link.

(Requires WP3 & PHP5)

== Installation ==

1. Unzip the downloaded plugin file `stardate.zip`,
2. Upload the folder `stardate` to the `/wp-content/plugins/` directory of your website,
3. Activate the plugin "Stardate" through the 'Plugins' menu in WordPress
4. That's it!  Look for the "Stardate" admin page under "Settings" to customize your set up and select which stardate format you want to use.
